## 서론 및 배경
현대 사회에서 자율주행과 교통 안전은 매우 중요한 이슈입니다.  
특히 교통 표지판은 운전자가 즉각적으로 인식해야 하는 핵심적인 시각 정보입니다.  
이 프로젝트는 **교통 표지판 인식을 통한 안전 보조 시스템**을 구현해봄으로써  
실제 교통 안전 기술에 기초적인 기여를 하고자 선택했습니다.

- 다양한 교통 표지판 이미지를 정확히 분류 (예: 제한 속도, 정지, 주의, 금지 표지 등)
